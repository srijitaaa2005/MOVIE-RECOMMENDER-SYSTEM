# MOVIE-RECOMMENDER-SYSTEM
A content-based movie recommender system built using NLP and machine learning techniques. The system recommends movies based on similarity in genres, cast, crew, and overview using the TMDB 5000 Movies dataset.

📌 Overview

This project recommends movies similar to a given input movie by analyzing textual metadata. It uses NLP techniques and cosine similarity to find and rank similar movies. The system is purely content-based and does not use user ratings.

⚙️ Features
Data preprocessing and feature engineering on TMDB dataset
NLP techniques: tokenization, stop-word removal, stemming
Feature extraction using Bag of Words (CountVectorizer)
Cosine similarity for computing movie similarity
Top-N movie recommendations based on similarity scores
🧠 Tech Stack

Python, Pandas, NumPy, Scikit-learn, NLTK

📂 Dataset

Uses the TMDB 5000 Movies Dataset (Kaggle):
https://www.kaggle.com/datasets/tmdb/tmdb-movie-metadata

⚠️ Dataset is not included due to large size. Users must download it separately and place it in the project folder.
